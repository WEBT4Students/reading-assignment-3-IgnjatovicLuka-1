Cheat Sheet

Html for Lists

1. Start tags for

  • Ordered list <ol>
  • Unordered list <ul>
  • Description list <dl/>
  • List item <li>
  • Description term <dt>
  • Description data <dd>

2. Css for Lists

  • Make bullet a square list-style-type: square;
  • Make bullet an image named "my-bullet.png" list-style-type: url('./my-bullet.png');
  • Make bullet an uppercase roman number list-style-type: upper-roman;
  • Set bullet position to be inside the list item's text flow list-style-position: inside
  • Set bullet position to be outside the list item's text flow list-style-position: outside

3. Html for Tables

1. Start tags for

  • Table <table>
  • Table row <tr>
  • Table header <thead/>
  • a table header cell <th>
  • a table body <tbody>
  • a table cell <td>
  • a table footer <tfoot>
  • a table footer cell <td>

4. Css for Tables

  • Black dotted border of 1 px width border: white 1px dotted;
  • Solid border of 1 px width having a red color and rounded corners border: red 1px solid; border-radius: 5px;
  • Merge the borders of table cells so that they appear as one line colspan="2";
